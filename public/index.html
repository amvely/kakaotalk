🔵 Meta API 설정 가이드

시스템 사용자(System User) 방식으로 발급된 토큰은 만료 기간이 없습니다.

💡 시스템 사용자 토큰 방식을 사용합니다. 개인 Facebook 계정과 독립적으로 동작하며, 한 번 발급 후 만료 걱정 없이 계속 사용할 수 있습니다.
STEP 1시스템 사용자 생성시스템 사용자 →
  1. business.facebook.com/settings/system-users 접속
  2. 추가 클릭 → 이름 입력 → 역할: Admin (관리자) 선택 후 저장
💡 Admin 선택 시 비즈니스 내 전체 광고 계정에 접근할 수 있습니다.
STEP 2광고 계정에 시스템 사용자 배정광고 계정 설정 →
  1. business.facebook.com/settings/ad-accounts 접속
  2. 조회할 광고 계정 선택 → 사용자 추가 클릭 → 생성한 시스템 사용자 선택
  3. 권한 선택 후 저장
●
성과 보기 권장보고서 조회 전용 — ads_read · read_insights 권한에 해당
○
광고 계정 관리광고 생성·수정까지 가능한 전체 권한 — 필요한 경우에만 선택
STEP 3Meta 앱 준비Meta for Developers →
  1. developers.facebook.com/apps 접속 → 앱 만들기 클릭
  2. 앱 유형: 비즈니스 선택 → 앱 이름 입력 후 저장
💡 이미 비즈니스 앱이 있다면 새로 만들지 않아도 됩니다.
STEP 4영구 토큰 발급만료없음시스템 사용자 →
  1. 시스템 사용자 설정으로 돌아와 생성한 사용자 선택 → 토큰 생성 클릭
  2. STEP 3에서 만든 앱 선택 → 권한에서 ads_read, read_insights 체크
  3. 발급된 토큰 (EAA...) 복사 → Access Token 입력란에 붙여넣기
✅ 시스템 사용자 토큰은 만료되지 않습니다. 한 번 발급 후 재발급 없이 계속 사용할 수 있습니다.
STEP 5광고 계정 ID 확인광고 계정 설정 →
  • 광고 계정 설정에서 계정 ID를 확인합니다.
  • 숫자만 입력하거나 act_123456789 형태로 입력해도 자동 처리됩니다.
⚠️ 토큰은 광고 데이터 접근 권한을 가진 민감 정보입니다. 외부에 공유하지 마세요.

여기에 입력된 계정도 실제 조회 대상에 포함됩니다. 협력광고 계정은 catalog_segment_actions / catalog_segment_value 기준으로 계산합니다.

🔴 Google MCC 공통 설정 1회 설정 · 공통 저장

MCC(관리자) 계정 기준으로 1회만 입력하면, MCC 하위 계정 조회에 공통 적용됩니다.

📋 MCC 최초 설정 순서
①
MCC 계정 생성 — Google Ads → 새 계정 만들기 → 관리자 계정 선택
②
하위 광고주 연결 — MCC → 계정 → 기존 계정 연결 (각 광고주 CID 필요)
③
GCP OAuth 앱 생성 — GCP 콘솔 →
  1. Google Ads API 사용 설정
  2. 사용자 인증 정보 → OAuth 2.0 클라이언트 ID 만들기
  3. 유형: 웹 애플리케이션 선택
  4. 승인된 리디렉션 URI에 https://developers.google.com/oauthplayground 추가
  5. Client ID / Client Secret 복사
④
Refresh Token 발급 — OAuth Playground →
  1. 우측 상단 ⚙ 클릭 → Use your own OAuth credentials 체크
  2. Client ID / Client Secret 입력 후 닫기
  3. 좌측 검색창에 아래 URL 입력 후 엔터:
    https://www.googleapis.com/auth/adwords
  4. Authorize APIs 클릭 → MCC 계정으로 로그인
  5. Exchange authorization code for tokens 클릭
  6. Refresh Token 복사 → 아래 입력란에 붙여넣기
⑤
Developer Token — MCC 계정 → 도구 및 설정 → API 센터 → 토큰 확인 (기본 엑세스 신청 필요) API 센터 →
⑥
아래 항목 입력 후 저장 — 이후 조회 대상 변경 시 CID만 입력하면 끝!
