API 연결 가이드

각 매체별 API Key/Token 발급 경로를 탭으로 확인하세요. 링크는 새 탭으로 열립니다.

네이버 검색광고 API Key 발급

네이버 검색광고 관리시스템SearchAd API 문서
  1. 네이버 검색광고 관리시스템에 로그인합니다.
  2. 상단 도구 또는 계정 메뉴에서 API 사용 관리 메뉴로 이동합니다.
  3. API 사용을 신청/동의한 뒤 Customer ID, Access License, Secret Key를 확인합니다.
  4. 이 대시보드의 API 설정 팝업에서 네이버 탭 항목에 각각 붙여넣습니다.
  5. 대행 계정은 조회 권한이 있는 광고주 계정의 Customer ID인지 확인하세요.
Secret Key는 외부 공유 금지 항목입니다. 실제 운영 배포에서는 브라우저 저장보다 서버 환경변수 저장을 권장합니다.

Meta Marketing API Access Token 발급

시스템 사용자광고 계정 설정Meta for Developers
  1. Business Settings에서 시스템 사용자를 생성하고 Admin 또는 필요한 조회 권한을 부여합니다.
  2. 조회할 광고 계정에 시스템 사용자를 추가하고 ads_read, read_insights 권한이 동작하도록 설정합니다.
  3. Meta for Developers에서 비즈니스 앱을 준비한 뒤 시스템 사용자 화면에서 토큰 생성을 클릭합니다.
  4. 권한에서 ads_read, read_insights를 선택하고 생성된 EAA... Access Token을 복사합니다.
  5. 광고 계정 ID는 act_123456789 형태 또는 숫자만 입력해도 됩니다. 여러 계정은 줄바꿈/쉼표로 입력하세요. 협력광고 계정은 API 설정의 “협력광고 Ad Account ID”에도 별도로 입력하세요.
개인 단기 토큰보다 시스템 사용자 토큰 방식을 권장합니다. 토큰 권한과 광고 계정 배정이 누락되면 성과 조회가 실패합니다.

Google Ads API OAuth/Token 발급

GCP Google Ads APIOAuth PlaygroundGoogle Ads
  1. Google Cloud Console에서 프로젝트를 선택하고 Google Ads API를 사용 설정합니다.
  2. API 및 서비스 → 사용자 인증 정보에서 OAuth 클라이언트 ID를 생성합니다. 데스크톱 앱 유형을 권장합니다.
  3. 생성된 Client ID와 Client Secret을 복사합니다.
  4. OAuth Playground 설정에서 Use your own OAuth credentials를 체크하고 Client ID/Secret을 입력합니다.
  5. 스코프에 https://www.googleapis.com/auth/adwords 를 추가하고 권한 승인 후 Refresh Token을 발급합니다.
  6. Google Ads 우측 상단의 Customer ID를 확인하고, MCC로 운영한다면 Login Customer ID와 Developer Token도 입력합니다.
Developer Token은 MCC 관리자 계정의 API 센터에서 확인합니다. 실계정 조회에는 Google Ads API 접근 승인이 필요할 수 있습니다.

네이버 검색광고

Meta Marketing API

비즈니스 포트폴리오 1개 안의 광고계정 2개는 줄바꿈/쉼표로 모두 입력하면 합산됩니다. Business ID만 입력한 경우에는 owned/client ad accounts를 자동 조회합니다.
협력광고 계정만 입력하세요. 여기에 입력된 계정도 실제 조회 대상에 포함됩니다. 협력광고 계정은 catalog_segment_actions / catalog_segment_value의 공유 전환 기준으로 계산합니다.
광고주별로 저장됩니다. 리드/문의 캠페인은 여기서 기준을 바꾼 뒤 저장하세요.
커스텀 선택 시 여러 값은 줄바꿈·쉼표·세미콜론으로 입력할 수 있습니다.

Google Ads API

실제 광고가 집행되는 하위 광고계정 ID를 입력하세요. 여러 개는 줄바꿈/쉼표로 입력할 수 있습니다. MCC만 입력한 경우 가능한 하위 계정 자동 조회를 시도합니다.
MCC로 권한을 위임받은 경우 관리자 계정 ID를 입력합니다.
